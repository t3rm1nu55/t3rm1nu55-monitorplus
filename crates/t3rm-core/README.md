# t3rm-core

Collector library for t3rm1nu55-monitorplus.

Defines the `Source` trait, privilege tier enum, and (Sprint 1+) the runtime channel-discovery layer and concrete metric sources for IOReport, SMC, sysinfo, and libproc.

**Status:** Sprint 0 scaffold. Public surface is stub APIs only.

See `CLAUDE.md` and `PLAN.md` at the repo root for architecture.
